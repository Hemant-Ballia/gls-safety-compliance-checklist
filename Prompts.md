# Prompts Used

I used ChatGPT as a helper while building this module. I used it mainly to understand the TRD, check my code against the requirements, and fix issues during testing.

Prompts I used:

1. Check whether my HTML structure matches the GLS safety checklist requirements.
2. Review my CSS and suggest small improvements without using any framework.
3. Check my JavaScript logic for localStorage, inspector session, task CRUD, filters, sorting, audit log, and JSON export.
4. Help me fix the dummy inspector names and baseline task text.
5. Review my README and make sure it explains the project clearly.

I tested the final project manually in the browser before submission.
